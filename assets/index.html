Adaptiv MCP – GitHub connector

This service exposes the MCP SSE endpoint and a small diagnostics UI.

  • MCP stream: /sse
  • Health check: /healthz
  • Tool discovery: /tools and /resources
  • Diagnostics JSON: /ui.json
  • Tool catalog UI: /ui/tools

If you are trying to connect a client, point it at https://<host>/sse.
