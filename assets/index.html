Adaptiv MCP GitHub

Lightweight diagnostics and discovery endpoints for the MCP server
Quick checks
/healthz
Service health status and baseline config signals
/tools
Tool catalog (names, descriptions, and optional schemas)
/resources
Resource catalog (URIs for tool invocation)
/ui.json
Small JSON descriptor of endpoints
/sse
MCP server stream endpoint
Operational notes
  • Responses default to Cache-Control: no-store for safety.
  • Static assets under /static/* are cacheable.
  • Use /tools for discovery; invoke with POST /tools/<name>.
